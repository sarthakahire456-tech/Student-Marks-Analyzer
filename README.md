# Student Marks Analyzer

## 📌 Project Overview
This project analyzes student marks using Python and NumPy. It calculates total, average, highest, and lowest marks and assigns grades based on performance.

## ⚙️ Tools Used
- Python
- NumPy

## 🔍 Features
- Accepts marks for multiple students
- Stores data in NumPy array
- Calculates total, average, max, and min marks
- Assigns grade using conditional logic

## 💡 Key Concepts
- Loops and conditions
- Functions
- NumPy operations

## 🚀 Conclusion
This project demonstrates basic data analysis using Python and NumPy while improving logical thinking and problem-solving skills.

## ▶️ How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Enter marks as input
